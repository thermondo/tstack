## Format

Each domain entry uses this shape so the aura can match cleanly:

```
### <domain-name>
**What:** one-line description.
**Why:** the business rationale — what changes if this works.
**Customer:** the human or segment served.
**Owner:** person or team (cross-link to PEOPLE.md if applicable).
**Status:** active | pre-launch | sunset.
```

Keep entries short. Auras surface the relevant block; deep context lives in the canonical strategy doc.
